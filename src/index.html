The TANGO-CC is now available as a Progressive Web App (PWA).

A PWA is a web application that behaves like a native app. The TANGO-CC can be installed on your device and run offline once it has cached all its resources. A little pop-up menu will inform you when the website is ready to run offline. Please note that you will need to enable the PWA again if you delete your browser cache.
